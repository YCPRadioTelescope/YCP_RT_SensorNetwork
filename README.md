# YCP_RT_SensorNetwork

This repository contains the Sensor Network code that is run on the Teensy 4.1 microcontroller embedded in the Radio Telescope.

### Devoplment setup for teensy 4.1 development

1. Install Visual Studio Code https://code.visualstudio.com/download
2. Open Visual Studio Code
3. Open Extensions Tab

![ESS-Dev-1](https://user-images.githubusercontent.com/51765760/151677857-4660bbd2-4339-43eb-bb6c-816ba0977f38.png)

4. Install C/C++ IntelliSense (should be one of most popular extensions) extension

![ESS-Dev-2](https://user-images.githubusercontent.com/51765760/151677868-8d56a3d3-5529-4b11-b0a2-e8d873bff77b.png)

5. Install PlatformIO extension

![ESS-Dev-3](https://user-images.githubusercontent.com/51765760/151677880-6b244773-c8f7-4acc-877a-fae05d1381f6.png)

6. Install Arduino https://www.arduino.cc/en/software 
7. Install Teensyduino https://www.pjrc.com/teensy/td_download.html
8. Clone Sensor Network Repository
    1. Open Visual Studio Code
    2. Select source control tab
    
    ![ESS-Dev-4](https://user-images.githubusercontent.com/51765760/151677893-ea8c7f27-d89d-4208-883a-2cc1c0232548.png)
    
    3. Select Clone Repository
    4. Enter sensor network URL into prompt https://github.com/YCPRadioTelescope/YCP_RT_SensorNetwork.git
        1. IMPORTANT: Place the repo in the following directory - C:\Users\%your user%\Documents\PlatformIO\Projects.
        You may need to create this directory if it does not already exist. This is where all PlatformIO Projects need to be saved.
